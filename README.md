# 🎮 WhiteBox: Transparent Gacha Adventure

**WhiteBox**에 오신 것을 환영합니다!
이 게임은 가챠(뽑기) 시스템에 블록체인 기술을 도입하여, 게임사의 확률 조작 걱정 없이 투명하고 공정하게 즐길 수 있는 **Web2.5 가챠 어드벤처 게임**입니다.

## 🛡️ 왜 이 게임인가요? (The "Transparent" Factor)

일반적인 게임과 달리, 여러분의 뽑기 결과는 게임사 서버 내부에만 머무르지 않습니다.

* **조작 불가:** 모든 뽑기 결과는 **Chainlink VRF**라는 검증된 외부 난수 시스템을 통해 결정되어, 개발자조차 확률을 조작할 수 없습니다.
* **결과 확인:** 여러분이 뽑은 결과는 블록체인에 영구 기록되어 이더스캔(Etherscan)을 통해 누구나 직접 확인할 수 있습니다.
* **편의성:** 복잡한 지갑 생성이나 암호화폐 지식 없이도, 기존 게임처럼 편하게 즐기실 수 있는 하이브리드(Web2.5) 시스템입니다.

---

## 🚀 게임 시작하기 (Quick Start)

게임을 플레이하기 위해 필요한 간단한 단계입니다.

### 1. 백엔드 서버 실행

게임이 블록체인과 통신하기 위한 다리 역할을 하는 서버를 실행해야 합니다.

1. 터미널을 열고 프로젝트의 `server` 폴더로 이동합니다.
2. 아래 명령어를 입력하여 서버를 켭니다.
```bash
node server.js

```


*(서버가 `http://localhost:3000`에서 실행 중인지 확인하세요!)*

### 2. 게임 실행

두 가지 방법 중 편한 것을 선택하세요.

* **방법 A: 웹 브라우저에서 실행 (WebGL)**
* 릴리즈에 올라가있는 파일을 받은 후 빌드된 웹 폴더에서 로컬 서버를 실행하여 브라우저로 접속합니다.


```bash
# 예시: 빌드 폴더에서 npx http-server 실행
npx http-server ./Builds

```


* 브라우저에 표시된 주소(예: `http://localhost:8080`)로 접속하세요.

* **방법 B: 유니티 에디터에서 실행**
* 유니티에서 프로젝트를 열고 `Play` 버튼을 누르면 서버와 연결되어 바로 플레이 가능합니다.
